**VariablesYOperadores**

Descripción del Proyecto



Este proyecto fue desarrollado en Java con el objetivo de aplicar el uso de variables, operadores y estructuras de control.

El programa permite realizar una clasificación de edad, identificar un día de la semana y verificar el acceso de un usuario mediante el uso de operadores lógicos.



 **Respuestas Teóricas**

¿Para qué se utilizan los operadores lógicos en programación?



Los operadores lógicos se utilizan para combinar condiciones y evaluar expresiones booleanas. Permiten tomar decisiones dentro de un programa mediante estructuras condicionales.

Los principales operadores lógicos son:



\&\& (AND): Devuelve verdadero si ambas condiciones son verdaderas.



|| (OR): Devuelve verdadero si al menos una condición es verdadera.



! (NOT): Niega el valor de una condición.



Estos operadores son fundamentales para el control del flujo del programa y la toma de decisiones.



¿Por qué es importante declarar correctamente el tipo de dato de una variable?



Es importante declarar correctamente el tipo de dato porque cada variable almacena información específica.

Si se utiliza un tipo incorrecto, pueden producirse errores en el programa o resultados inesperados.



Por ejemplo:



int se usa para números enteros.



double para números decimales.



String para texto.



boolean para valores verdadero o falso.



Una correcta declaración mejora la seguridad, eficiencia y claridad del código.



 **Justificación Técnica**

Clasificación de edad



Para la clasificación de edad se utilizó la estructura condicional if - else if - else, ya que permite evaluar múltiples rangos de valores y ejecutar diferentes bloques de código dependiendo de la condición que se cumpla.



Esta estructura es adecuada cuando se deben comparar varios intervalos numéricos.



Día de la semana



Para el día de la semana se utilizó la estructura switch, porque permite evaluar una variable con múltiples posibles valores específicos de manera más organizada y legible que múltiples if.



El uso de switch mejora la claridad del código cuando se trabaja con opciones definidas.



Verificación de acceso



Para la verificación de acceso se utilizaron operadores lógicos (\&\&) junto con una estructura if, ya que era necesario evaluar más de una condición al mismo tiempo.



Esto permite validar que ambas condiciones sean verdaderas para conceder el acceso.



**Ejecución del Programa**



El programa se ejecuta desde la clase Main y no presenta errores de compilación.

Todo el código se encuentra organizado dentro de la carpeta src.
